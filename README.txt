--Java-библиотека Directed Acyclic Graphs--
-Разработчик: Ксения Лёгостева БПИ208-

Эта библиотека представляет собой направленный ацикличный граф в двумерном пространстве.

Классы:
-BoundBox (ограничивающий)
-Coord2D (координаты)
-DAGConstraintException
-Origin (система координат со смещением)
-Point (физический точки)
-Space (мировая система координат)
