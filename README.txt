--Java-библиотека Directed Acyclic Graphs--

Направленный ацикличный граф в двумерном пространстве.

Классы:
- Space (оосновная система координат)
- Origin (система координат со смещением)
- Point (физическая точка)
- Coord2D (координата точки)
- BoundBox (ограничивающий прямоугольник, в который попадают все физические точки относительно некоторой системы координат)
